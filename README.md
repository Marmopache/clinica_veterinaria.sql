# clinica_veterinaria.sql
Script SQL para la creación de la base de datos de una clínica veterinaria. Incluye la definición de tablas (Dueño, Mascota, Profesional, Atención), inserción de datos de ejemplo, consultas básicas, actualización y eliminación de registros, así como transacciones para operaciones combinadas.

# Clínica Veterinaria - Base de Datos SQL

**Autora:** Gabriela de Jesús Alvarado Rosales  
**Módulo:** Bases de Datos Relacionales  
**Lenguaje:** SQL  

## Descripción
Este repositorio contiene un script SQL para la creación de la base de datos de una **clínica veterinaria**. Incluye:

- Creación de la base de datos y tablas:
  - `Dueno`
  - `Mascota`
  - `Profesional`
  - `Atencion`
- Inserción de datos de ejemplo.
- Consultas básicas para obtener información de dueños, mascotas y atenciones.
- Actualización y eliminación de registros.
- Uso de **transacciones** para operaciones combinadas (insertar, actualizar y eliminar datos de manera segura).

Este material corresponde al módulo de **Bases de Datos Relacionales** del bootcamp **Desarrollo de Aplicaciones Full Stack Python**.

## Estructura del Script

1. **Crear base de datos**  
2. **Crear tablas** con claves primarias y foráneas.  
3. **Insertar datos de ejemplo** en cada tabla.  
4. **Consultas SQL** para extraer información relevante.  
5. **Actualizar y eliminar registros**.  
6. **Transacciones** para manejar operaciones complejas de manera segura.
